Сравнение методов хранения данных

Гипервизоры

  • Выделенное место недоступно
  • Не так уж просто выдернуть данные из виртуалки
  • Есть пенальти из-за полного образа системы

Docker

  • Образы наслаиваются друг на друга
  • Реальный размер контейнера ничтожно мал
  • Best practice: хранение данных на хост машине, доступ к данным при помощи томов
  • Том - может быть и файлом и директорией
